HOME RUN LAB — GITHUB PAGES UPLOAD

Upload every file in this folder to your GitHub repository root.
Do not upload this folder itself; upload the contents inside it.

Required files:
- index.html
- slate.json
- hr-history.json
- live-events.json
- manifest.webmanifest
- sw.js
- icon.svg

After uploading:
1. Go to GitHub repo Settings > Pages.
2. Source: Deploy from branch.
3. Branch: main.
4. Folder: / root.
5. Save.

On iPhone:
1. Open the GitHub Pages URL in Safari.
2. Tap Share.
3. Tap Add to Home Screen.
